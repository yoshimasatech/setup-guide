# Setup Guide

クライアント向けの各種セットアップガイドをまとめたリポジトリです。

## 内容

- **Firebase プロジェクト作成ガイド** — Firebase の初期設定手順（PDF / HTML）
- **Apple Developer Program 登録ガイド** — Apple Developer Programへの個人登録と開発者招待の手順（PDF / HTML）
- **APIキー 取得・設定ガイド** — OpenAI / Anthropic / Google のAPIキー取得と共有方法（PDF / HTML）

## 用途

開発案件でクライアントに環境構築をお願いする際のガイド資料を一元管理しています。
