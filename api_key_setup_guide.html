APIキー 取得・設定ガイド

OpenAI / Anthropic (Claude) / Google / App Store Connect のAPIキー取得と開発者への共有方法
作成日: 2026年5月9日
じょん総合研究所
APIキーとは: AIサービスをアプリに組み込むための「鍵」のようなものです。お客様のアカウントでキーを発行し、開発者にお伝えいただくことで、お客様名義のアカウントでサービスを利用できるようになります。

費用について: APIキーの発行自体は無料です。実際にアプリからAIを呼び出した回数・量に応じて従量課金されます。利用上限の設定も可能です。

Part 1: OpenAI（ChatGPT）のAPIキー取得

ChatGPTやGPT-4などのAI機能をアプリに組み込む場合に必要です

1
OpenAI Platform にアクセス
ブラウザで platform.openai.com にアクセスします。
アカウントをお持ちでない場合は 「Sign up」 からアカウントを作成してください。
メールアドレスまたはGoogleアカウントで登録できます。
2
支払い情報を登録
ログイン後、左側メニューの 「Settings」→「Billing」 をクリックします。
「Add payment details」 からクレジットカード情報を登録します。

初回クレジットの購入画面が表示されるので、最低金額（$5〜） でチャージしてください。
APIの利用料金はチャージ（前払い）方式です。残高がなくなるとAPIが停止するため、意図しない高額請求の心配はありません。
3
利用上限を設定（推奨）
「Settings」→「Limits」 から、月間の利用上限を設定できます。
「Set a monthly budget」 に上限金額（例: $10）を入力し、保存してください。
4
APIキーを発行
左側メニューの 「API keys」 をクリックします。
「Create new secret key」 をクリックし、名前を入力します（例: 「my-app」など、自由に設定できます）。

「Create secret key」 をクリックすると、sk- で始まるキーが表示されます。
重要: 表示されたキーは この画面でしか確認できません。必ずコピーして安全な場所に保存してください。
画面を閉じると同じキーは二度と表示されません（再発行は可能です）。
5
開発者にキーを共有
コピーしたAPIキーを開発者にお伝えください。
共有方法は、チャットのメッセージで直接お送りいただければ大丈夫です。

Part 2: Anthropic（Claude）のAPIキー取得

Claude AIの機能をアプリに組み込む場合に必要です

1
Anthropic Console にアクセス
ブラウザで console.anthropic.com にアクセスします。
アカウントをお持ちでない場合は 「Sign up」 からアカウントを作成してください。
メールアドレスまたはGoogleアカウントで登録できます。
2
支払い情報を登録
ログイン後、左側メニューの 「Settings」→「Billing」 をクリックします。
「Add payment method」 からクレジットカード情報を登録します。

クレジットの購入画面で 最低金額（$5〜） をチャージしてください。
OpenAIと同様にチャージ（前払い）方式です。残高がなくなるとAPIが停止します。
3
利用上限を設定（推奨）
「Settings」→「Limits」 から、月間の利用上限（Spend limit）を設定できます。
上限金額（例: $10）を入力し、保存してください。
4
APIキーを発行
左側メニューの 「API Keys」 をクリックします。
「Create Key」 をクリックし、名前を入力します（例: 「my-app」など）。

「Create Key」 をクリックすると、sk-ant- で始まるキーが表示されます。
重要: 表示されたキーは この画面でしか確認できません。必ずコピーして安全な場所に保存してください。
5
開発者にキーを共有
コピーしたAPIキーを開発者にお伝えください。
共有方法は、チャットのメッセージで直接お送りいただければ大丈夫です。

Part 3: Google（Gemini）のAPIキー取得

Gemini AIの機能をアプリに組み込む場合に必要です

1
Google AI Studio にアクセス
ブラウザで aistudio.google.com にアクセスし、Googleアカウントでログインします。
2
APIキーを発行
左側メニューの 「Get API key」 をクリックします。
「Create API key」 をクリックし、プロジェクトを選択（または新規作成）します。

キーが生成されるので、コピーして安全な場所に保存してください。
Google Gemini APIは無料枠が大きく、小規模な利用であれば無料で利用できる場合があります。
有料プランに移行する場合は、Google Cloud Consoleで請求先アカウントの設定が必要です。
3
開発者にキーを共有
コピーしたAPIキーを開発者にお伝えください。
共有方法は、チャットのメッセージで直接お送りいただければ大丈夫です。

Part 4: App Store Connect APIキーの発行

iOSアプリの審査提出・管理を開発者に委任する場合に必要です

この方法が向いている方:
Apple Developerアカウントのログイン情報を開発者に直接渡したくない場合に、APIキーを発行して共有する方法です。
手間は少し増えますが、アカウントのセキュリティを保ちながら開発者にアプリの審査提出や管理を委任できます。

※ 開発者をApp Store Connectに「ユーザー」として招待する方法もあります（Apple Developer登録ガイドのPart 3を参照）。
どちらの方法が良いかは、開発者にご相談ください。
1
App Store Connect にログイン
ブラウザで appstoreconnect.apple.com にアクセスし、Apple IDでログインします。
2
「ユーザーとアクセス」を開く
トップページのメニューから 「ユーザーとアクセス」 をクリックします。
3
「統合」タブ → 「App Store Connect API」を選択
上部のタブから 「統合」 をクリックし、左側メニューの 「App Store Connect API」 を選択します。

初回アクセス時は「APIキーへのアクセスをリクエスト」ボタンが表示される場合があります。
その場合はボタンをクリックし、利用規約に同意してください。
4
APIキーを生成
「+」ボタン（キーを生成） をクリックします。

以下を入力・選択してください：
  • 名前: 任意の名前（例: 「開発者用」など）
  • アクセス: 「Admin」 を選択

入力後、「生成」 をクリックします。
「Admin」権限を選択すると、開発者がアプリの登録・ビルドのアップロード・審査提出など、すべての操作を行えるようになります。
5
APIキーファイル (.p8) をダウンロード
キーが生成されると、一覧に新しいキーが表示されます。
「APIキーをダウンロード」 をクリックして、.p8 ファイルをダウンロードしてください。
重要: .p8ファイルは 一度しかダウンロードできません。
必ずダウンロードし、安全な場所に保管してください。紛失した場合は、キーを削除して再発行する必要があります。
6
Key ID と Issuer ID を確認
同じ画面に以下の2つのIDが表示されています：
  • Key ID: キー一覧の各行に表示されています（例: ABC1234DEF）
  • Issuer ID: ページ上部に表示されています（例: 12345678-abcd-efgh-ijkl-123456789012）

これらもコピーしておいてください。
7
開発者に3点セットを共有
以下の 3つ を開発者にお送りください：
  • .p8 ファイル（ダウンロードしたファイル）
  • Key ID
  • Issuer ID

チャットのメッセージで直接お送りいただければ大丈夫です。
この3つが揃うことで、開発者はお客様のアカウントにログインせずに、アプリの審査提出やビルド管理を行えるようになります。

APIキー取得の流れ（共通）

  • 各サービスのサイトにアクセスし、アカウントを作成する
  • 支払い情報（クレジットカード）を登録する
  • 利用上限を設定する（推奨）
  • APIキーを発行し、コピーして開発者に共有する
セキュリティに関するご注意:
・APIキーは「パスワード」のようなものです。第三者に漏洩しないようご注意ください
・SNSやブログなどに公開しないでください
・万が一漏洩した場合は、各サービスの管理画面からキーを無効化（削除）し、新しいキーを再発行してください
どのAPIキーが必要かは開発者からご案内します。
すべてのキーが必要とは限りません。開発するアプリの要件に応じて、必要なサービスのキーのみ取得していただきます。
ご不明な点がございましたら、お気軽にご連絡ください。
画面共有でのサポートも可能です。
