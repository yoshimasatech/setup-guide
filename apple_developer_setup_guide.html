Apple Developer Program
登録ガイド

Apple ID作成 → Developer Program登録 → 開発者への招待
作成日: 2026年5月9日
じょん総合研究所

Part 1: 事前準備

登録に必要なもの・環境を確認します

1
必要なものを確認
以下のものをご用意ください。
  • iPhone または iPad（Touch ID / Face ID / パスコードが有効なもの）
  • 本人確認書類（パスポートまたは運転免許証）
  • クレジットカードまたはデビットカード（年会費の支払い用）
  • Apple ID（お持ちでない場合は次のステップで作成します）
登録手続きはiPhone/iPadの「Apple Developer」アプリから行います。
PCからは登録できませんのでご注意ください。
2
Apple ID の2ファクタ認証を有効にする
Apple Developer Programへの登録には、2ファクタ認証が有効なApple ID が必要です。

確認方法: 設定 → [自分の名前] → サインインとセキュリティ → 2ファクタ認証
「オン」になっていればOKです。オフの場合は画面の指示に従って有効にしてください。
3
Apple ID の名前を本名（法的氏名）にする
Apple IDに登録されている 姓・名 は、App Storeで「販売者名」として公開されます。
必ず 本名（法的氏名） になっていることを確認してください。

確認方法: 設定 → [自分の名前] → 名前、電話番号、メール
注意: ニックネームや会社名をApple IDの名前に設定していると、登録審査が遅れる場合があります。

Part 2: Apple Developer アプリからの登録

iPhone/iPadを使ってApple Developer Programに登録します

1
「Apple Developer」アプリをインストール
App Storeで 「Apple Developer」 と検索し、Appleの公式アプリをインストールします。
アプリを開き、Apple IDでサインインします。
2
「今すぐ登録」をタップ
アプリの「アカウント」タブを開き、「Apple Developer Programに登録」 をタップします。
続いて 「今すぐ登録」 をタップして登録手続きを開始します。
3
登録タイプを選択
登録タイプの選択画面が表示されます。
「個人 / 個人事業主」 を選択してください。
法人（会社）として登録する場合は「組織」を選択しますが、D-U-N-S番号が必要になります。
個人アプリの場合は「個人」で問題ありません。
4
本人確認
画面の指示に従い、本人確認を行います。
パスポートまたは運転免許証 をカメラで撮影し、顔認証（セルフィー）を行います。
本人確認はAppleによる自動処理で行われます。書類がはっきり読み取れるよう、明るい場所で撮影してください。
5
連絡先情報の入力
以下の情報を入力します。
  • メールアドレス（連絡用）
  • 電話番号
  • 住所
6
ライセンス契約に同意
Apple Developer Program ライセンス契約の内容を確認し、「同意する」 をタップします。
7
年会費の支払い
年会費 12,980円（税込） の支払いを行います。
Apple IDに登録されている支払い方法（クレジットカード等）で決済されます。

「購読」 をタップして支払いを完了します。
年会費について: Apple Developer Programは年間サブスクリプションです。
1年後に自動更新されます（不要な場合は更新前にキャンセル可能）。
8
審査完了を待つ
支払い完了後、Appleによる審査が行われます。
通常 24時間〜数日 で審査が完了し、登録メールアドレスに通知が届きます。

審査完了後、Apple Developerアプリまたは developer.apple.com にログインすると、メンバーシップが有効になっています。
審査に時間がかかる場合（1週間以上）は、Appleサポートに問い合わせることも可能です。

Part 3: 開発者を招待する

App Store Connectで開発者にアクセス権限を付与します

1
App Store Connect にログイン
PCまたはスマートフォンのブラウザで appstoreconnect.apple.com にアクセスし、Apple IDでログインします。
2
「ユーザーとアクセス」を開く
トップページのメニューから 「ユーザーとアクセス」 をクリックします。
3
新しいユーザーを追加
「+」ボタン をクリックし、以下の情報を入力します。
  • 姓名: 開発者の名前
  • メールアドレス: 開発者のApple ID（メールアドレス）
  • 役割: 「Admin」を選択
開発者のメールアドレスは別途お伝えします。
「Admin」権限を付与すると、アプリの登録・審査提出・証明書管理などすべての操作が可能になります。
4
招待を送信
入力内容を確認し、「招待」 をクリックします。
開発者のメールアドレスに招待メールが届き、承認すると参加が完了します。

全ての設定が完了しました

  • Apple Developer Programに登録されました
  • 開発者がAdmin権限で招待されました
  • この後の技術的な設定（証明書・プロファイル作成、アプリ登録など）は開発者が行います
費用まとめ:
・Apple Developer Program 年会費: 12,980円（税込）/ 年
・それ以外の費用は発生しません
ご不明な点がございましたら、お気軽にご連絡ください。
画面共有でのサポートも可能です。
